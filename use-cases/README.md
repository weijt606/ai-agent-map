# Use Cases

中文：
这个目录从“真实问题”出发组织内容，而不是从 agent 名称出发。对于大多数读者，这应该是选型时的第一站。

English:
This directory is organized around real problems, not agent names. For most readers, this should be the first stop when choosing an agent.

## 这个目录放什么 | What Belongs Here

- 某一类问题的选型指南。 Selection guides for a concrete problem class.
- 同一问题下的推荐 agent 与取舍逻辑。 Recommended agents and trade-offs within that problem class.
- 面向新手的简化决策路径。 Beginner-friendly decision paths.
